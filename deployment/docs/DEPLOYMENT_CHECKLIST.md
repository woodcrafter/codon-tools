# 部署检查清单

使用本清单确保部署过程中的每个步骤都已正确完成。

## 部署前准备

### 服务器准备
- [ ] 服务器已准备就绪（Ubuntu 20.04/22.04）
- [ ] 拥有root或sudo权限
- [ ] 服务器可以访问互联网
- [ ] 记录服务器公网IP地址: __________________

### 域名配置
- [ ] 域名 dreamstudios.work 已注册
- [ ] DNS A记录已配置（@ → 服务器IP）
- [ ] DNS A记录已配置（www → 服务器IP）
- [ ] DNS解析已生效（使用 `ping dreamstudios.work` 验证）

### 项目文件
- [ ] 项目文件已下载或克隆
- [ ] 部署脚本已准备（deployment目录）

### 第三方服务（可选）
- [ ] 钉钉应用已创建（如需钉钉登录）
- [ ] 钉钉AppKey和AppSecret已获取
- [ ] 钉钉Webhook已配置（如需消息推送）

## 部署步骤

### 1. 环境安装
- [ ] 已连接到服务器
- [ ] 已上传项目文件到 `/var/www/protein-production-platform`
- [ ] 已执行 `deploy.sh` 脚本
- [ ] Node.js安装成功（验证: `node -v`）
- [ ] PostgreSQL安装成功（验证: `systemctl status postgresql`）
- [ ] Nginx安装成功（验证: `systemctl status nginx`）
- [ ] PM2安装成功（验证: `pm2 -v`）

### 2. 数据库配置
- [ ] 数据库已创建（gene_synthesis_db）
- [ ] 数据库用户已创建（gene_app_user）
- [ ] 数据库密码已记录（位置: `/root/db_credentials.txt`）
- [ ] 数据库连接测试成功

### 3. 环境变量配置
- [ ] `.env` 文件已生成
- [ ] DATABASE_URL已配置
- [ ] JWT_SECRET已生成
- [ ] 钉钉配置已添加（如需要）
- [ ] 其他必要环境变量已配置

### 4. 应用构建
- [ ] 依赖安装完成（`pnpm install`）
- [ ] 数据库迁移完成（`pnpm db:push`）
- [ ] 种子数据导入完成（`init-database.sh`）
- [ ] 项目构建成功（`pnpm build`）
- [ ] dist目录已生成

### 5. 应用启动
- [ ] PM2配置文件已复制
- [ ] 应用已启动（`pm2 start ecosystem.config.js`）
- [ ] PM2进程状态正常（`pm2 status`）
- [ ] 应用日志无错误（`pm2 logs`）
- [ ] 应用监听3000端口（`netstat -tuln | grep 3000`）

### 6. Nginx配置
- [ ] Nginx配置文件已安装
- [ ] Nginx配置测试通过（`nginx -t`）
- [ ] Nginx已重启（`systemctl restart nginx`）
- [ ] HTTP访问正常（http://dreamstudios.work）

### 7. SSL证书
- [ ] 域名解析已确认
- [ ] SSL证书申请成功（`setup-ssl.sh`）
- [ ] HTTPS访问正常（https://dreamstudios.work）
- [ ] HTTP自动重定向到HTTPS
- [ ] SSL证书自动续期已配置

### 8. 开机自启动
- [ ] PM2开机自启动已配置（`pm2 startup`）
- [ ] PM2进程列表已保存（`pm2 save`）
